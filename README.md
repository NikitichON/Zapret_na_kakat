# Мое простое задание))

- Что бы скачать YML нужно ввести команду:
- wget https://raw.githubusercontent.com/NikitichON/Zapret_na_kakat/main/docker-compose.yml
- После находясь в одном каталоге с файлом docker-compose.yml ввести
- docker-compose up -d
- Запустится контейнер с простым сайтом на nginx, перейти на него можно будет введя в строке браузера IP адрес устройства где находиться контейнер + порт 8080
- Например http://172.28.56.200:8080/
- Или можно ввести http://localhost:8080 находясь на том же устройстве где и работает docker
